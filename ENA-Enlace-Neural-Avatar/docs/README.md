## Como Empezar

Si eres nuevo en ENA, te recomendamos seguir este orden:

1.  Lee [`casos-uso.md`](casos-uso.md) para identificar que aplicacion se adapta a ti
2.  Sigue [`guia-instalacion.md`](guia-instalacion.md) para poner el sistema en marcha
3.  Usa [`manual-usuario.md`](manual-usuario.md) para aprender a usarlo dia a dia
4.  Consulta [`validacion-clinica.md`](validacion-clinica.md) si necesitas respaldo cientifico

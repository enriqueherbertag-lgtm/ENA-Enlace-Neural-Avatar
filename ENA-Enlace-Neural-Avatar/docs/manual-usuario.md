# Manual de Usuario - ENA (Enlace Neural con Avatar)

Guia paso a paso para usar el sistema ENA.

---

## 1. Que es ENA

ENA es un sistema que permite controlar un avatar digital usando solo tu mente. Esta disenado para personas con movilidad reducida, paralisis o ELA, ofreciendo una forma de comunicacion e interaccion con el entorno.

No requiere cirugia ni implantes. Usa un casco EEG comercial que lee tu actividad cerebral y la traduce en comandos.

---

## 2. Componentes del Sistema

### 2.1 Hardware incluido

- Casco EEG (NeuroSky MindWave 2)
- Raspberry Pi 4 (unidad de procesamiento)
- Cable de alimentacion
- Manual de instrucciones

### 2.2 Hardware necesario (no incluido)

- Monitor HDMI o TV
- Teclado y mouse (para configuracion inicial)
- Conexion a internet (opcional)

---

## 3. Configuracion Inicial

### 3.1 Desempaquetado

1. Retire con cuidado todos los componentes de la caja
2. Conecte la Raspberry Pi a la corriente electrica
3. Encienda el monitor y conectelo a la Raspberry Pi

### 3.2 Encendido del sistema

1. Presione el boton de encendido de la Raspberry Pi
2. Espere a que aparezca la pantalla de inicio (30-60 segundos)
3. Vera el escritorio de ENA con el icono del programa

### 3.3 Encendido del casco EEG

1. Presione el boton lateral del casco NeuroSky por 2 segundos
2. La luz del casco comenzara a parpadear en azul
3. Espere a que la luz se vuelva azul fijo (conexion establecida)

---

## 4. Uso Basico del Sistema

### 4.1 Iniciar la aplicacion

1. Haga doble clic en el icono "ENA" del escritorio
2. Aparecera la pantalla de inicio con el avatar 3D
3. El sistema mostrara "Conectado" cuando detecte el casco

### 4.2 Calibracion inicial (primera vez)

1. Sientese comodo y relajado
2. Mire fijamente el punto en la pantalla
3. Siga las instrucciones: "Relajese" (30 segundos), "Concentrese" (30 segundos)
4. El sistema aprendera sus patrones cerebrales

### 4.3 Modo de uso diario

Una vez calibrado, puede usar estos comandos mentales:

| Comando Mental | Que hace | Como lograrlo |
|:---|:---|:---|
| Concentracion | Mueve el avatar hacia adelante | Piense en un objeto fijo, enfoquese |
| Relajacion | Detiene el avatar | Respire profundo, libere la mente |
| Imaginacion izquierda | Gira el avatar a la izquierda | Imagine que mueve la mano izquierda |
| Imaginacion derecha | Gira el avatar a la derecha | Imagine que mueve la mano derecha |
| Parpadeo fuerte | Selecciona / confirma | Parpadee dos veces seguidas |

## 5. Aplicaciones Especificas

### 5.1 Modo Comunicacion (para personas con ELA)

Este modo permite escribir mensajes seleccionando letras en pantalla:

1. En el menu principal, seleccione "Comunicacion"
2. Aparecera un teclado en pantalla
3. Las letras se iluminan una por una
4. Cuando la letra deseada se ilumine, concentrese para seleccionarla
5. Las palabras formadas aparecen en la parte superior

**Velocidad esperada:** 3-5 letras por minuto (con practica)

### 5.2 Modo Control de Silla de Ruedas

Para usuarios que usan ENA con silla de ruedas electrica:

1. Conecte el cable de control a la silla
2. Seleccione "Modo Silla" en el menu
3. Use los comandos de imaginacion para dirigirse
4. El sistema tiene parada automatica de seguridad

### 5.3 Modo Rehabilitacion (post-ACV)

Ejercicios guiados para recuperar movilidad:

1. Seleccione "Rehabilitacion"
2. Elija el nivel (basico, intermedio, avanzado)
3. Siga las instrucciones: "Imagine que mueve el brazo"
4. El avatar mostrara el movimiento imaginado
5. Complete 10 repeticiones por sesion

## 6. Mantenimiento y Cuidados

### 6.1 Casco EEG

- Limpie los electrodos con alcohol antes de cada uso
- Cargue el casco cada 3-4 usos
- Guarde en lugar seco y fresco

### 6.2 Raspberry Pi

- Mantenga el equipo en lugar ventilado
- No desconecte mientras se esta apagando
- Actualice el software periodicamente

---

## 7. Solucion de Problemas Comunes

| Problema | Causa probable | Solucion |
|:---|:---|:---|
| El casco no conecta | Bateria baja | Cargue el casco 1 hora |
| | Bluetooth desactivado | Active Bluetooth en la Raspberry |
| El avatar no responde | Mala colocacion del casco | Ajuste el electrodo en la frente |
| | Usuario distraido | Relajese y concentrese |
| Respuestas lentas | Interferencia electrica | Aleje otros dispositivos |
| | Calibracion desactualizada | Repita calibracion |
| El sistema se congela | Sobrecalentamiento | Apague 10 minutos, mejore ventilacion |

## 9. Precauciones de Seguridad

- No use el sistema si tiene epilepsia fotosensible
- Tome descansos cada 30 minutos
- Detenga el uso si siente mareos o molestias
- Mantenga fuera del alcance de niños pequenos
